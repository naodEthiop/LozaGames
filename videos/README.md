# Video Files

Place your color action video files in this folder.

## Required Videos

The following video files should be added:

- `green.mp4` - Green color action (Run!)
- `red.mp4` - Red color action (Stop!)
- `yellow.mp4` - Yellow color action (Sit Down!)
- `black.mp4` - Black color action (Touch Your Hair!)
- `white.mp4` - White color action (Touch Your Teeth!)
- `blue.mp4` - Blue color action (Point to the Sky!)
- `pink.mp4` - Pink color action (Strike a Pose!)
- `purple.mp4` - Purple color action (Dance!)

## Video Requirements

- **Format**: MP4 (H.264 codec recommended)
- **Duration**: 5 seconds (configurable in `src/data/colors.json`)
- **Resolution**: 1920x1080 or 1280x720 recommended
- **Orientation**: Landscape preferred for TV display
- **Content**: Show the action being performed with the color theme

## Tips

- Keep videos short and engaging for kids
- Use bright, colorful backgrounds matching each color
- Show clear demonstrations of each action
- Consider adding background music or sound effects
- Ensure good lighting and visibility

## Fallback Behavior

If a video file is missing or fails to load, the app will display an animated gradient background with the color theme instead.
